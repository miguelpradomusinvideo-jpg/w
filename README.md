# 🎮 Rockstar 2FA Generator - Gg Community

Bot de Discord com botão interativo para gerar códigos 2FA para contas Rockstar Games.

## 👑 Créditos
**Desenvolvido por Miguel®**

## 🎮 Como usar

1. **Inicie o bot** e use o comando `!start` ou `/start`
2. **Clique no botão** "Gerar Código 2FA"
3. **Insira sua chave 2FA** no modal que aparece
4. **Receba seu código** de autenticação temporário

## 📱 Comandos

### Texto (prefixo `!`)
- `!start` ou `!menu` - Mostrar o menu com o botão
- `!ajuda` - Mostrar ajuda
- `!creditos` - Ver créditos

### Slash Commands
- `/start` - Mostrar o menu com o botão
- `/creditos` - Ver créditos

## 🔐 Onde encontrar minha chave 2FA?

1. Acesse sua conta Rockstar Games
2. Vá em **Configurações** > **Autenticação de Dois Fatores**
3. Copie a **chave secreta** (formato Base32)
4. Cole no modal que aparece ao clicar no botão
